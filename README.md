# SentinelAI

## AI-Driven Cyber Resilience Platform for Critical National Infrastructure

SentinelAI is an Agentic AI-powered cybersecurity platform that detects behavioral anomalies, maps attack progression using MITRE ATT&CK, correlates threat intelligence, and enables autonomous incident response.

### Key Features
- Behavioral Anomaly Detection
- Threat Intelligence Correlation
- MITRE ATT&CK Mapping
- Attack Prediction
- Autonomous Incident Response
- Security Knowledge Graph

### Technology Stack
- React.js
- Node.js
- MongoDB
- Python
- Scikit-Learn
- LangChain

### Architecture
Data Sources → Data Ingestion → Agentic AI Engine → Knowledge Graph → SOC Dashboard → Automated Response
